Software engineer vocabulary

Introduction

In this article I'm going to share with you a lot of software engineering vocabulary.

1. Immutability

An object is immutable if you can't modify its state after creation

2. Class Invariant

A property that always remains true for all instances of a class no matter what happens
Let's see the following example:

All the instances of the class Person will have a non null valid age.
If the dateOfBirth parameter is invalid, the instance will not be created.
